# ☕ Cashlo AI Academy

An interactive, gamified web app for learning AI/ML from the ground up — built to teach myself the foundations while developing my semester-5 research project: a **self-improving cash-flow forecasting system** for my coffee-shop POS ([Cashlo](https://github.com/)).

> Every concept is taught through a real coffee shop's data, so the theory always connects to something concrete.

## ✨ Features

- **3 levels** — Beginner → Intermediate → Advanced, unlocking as you progress
- **Chapter 0: The Math of AI** — mean & spread, correlation, line-fitting, gradient descent, taught with intuition first
- **Interactive widgets** — e.g. a drag-to-fit regression line where you watch the error shrink in real time
- **Kahoot-style quizzes** — instant feedback with explanations
- **Streak system** — a daily habit ladder (Small → Medium → Hard) inspired by *Atomic Habits*
- **Career Roadmap** — a 7-phase path to becoming a professional AI / ML / Prompt Engineer
- **Progress saving** — XP, streaks, and completion persist locally (no backend needed)

## 🛠️ Tech

Vanilla **HTML + CSS + JavaScript** — zero build step, zero dependencies. Progress is stored in `localStorage`. A custom "Coffee Lab" design system (warm, editorial, balanced palette).

## 🚀 Run locally

Just open `index.html` in a browser, or serve the folder:

```bash
python -m http.server 5500
# then visit http://localhost:5500
```
